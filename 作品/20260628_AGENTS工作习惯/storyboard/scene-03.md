# Scene 03｜0:30-1:05 厨房使用说明书

这一段是全片最适合连续插画化的部分：把 AGENTS.md 类比成厨房使用说明书。

## 分镜节奏

1-2：右侧标题 `厨房规则`，卡通小人站厨房门口。  
3-4：米缸和盐盒弹出，问题：`米在哪？`、`盐在哪？`。  
5-6：冰箱打开，分成 `今晚吃` 和 `先别动`。  
7-8：两瓶酱油对比，问题：`用哪瓶？`。  
9-10：炒锅和汤锅对比，问题：`炒锅 / 汤锅`。  
11-12：吃完后的碗筷和厨房台面。  
13-14：垃圾袋和门口，问题：`走前带走？`。  
15-16：所有规则合成一张 `AGENTS.md 厨房使用说明书`。
