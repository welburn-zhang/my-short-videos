# 语义分镜 v2｜AGENTS.md 是什么

这版不再按“每秒一个分镜”切，而是按脚本语义切镜头。一个画面可以停留 5-10 秒，只在信息关系发生变化时换图。

## 统一画风

- 横屏 16:9。
- 真人画面来自实拍，不让模型生成具体真人脸。
- 图像只承担右侧演示卡、贴纸场景、全屏录屏标注、收口总结卡。
- 色调保持：暖黄色室内光、米白信息板、橙色描边、深灰粗体字。
- 后期文字尽量重新叠加，生图只负责布局和视觉方向。

## 剪辑原则

1. 能用同一张图承载的信息，不拆成两张。
2. 只有“语义关系变了”才切图，比如从厨房类比切到文件夹迷宫。
3. 右侧贴纸可以在同一张底图上逐个弹出，不需要每弹一个贴纸就换一张新图。
4. 重复构图只保留一张，比如“先看规矩”不用连续两格，只做一次，然后靠后期缩放/弹出动画完成节奏。
