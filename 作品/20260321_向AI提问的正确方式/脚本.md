## 封面图设计

### 封面拍摄/制作指南


| 要素     | 具体要求                                          |
| ------ | --------------------------------------------- |
| **表情** | 眉头微蹙，眼神带一点"你确定你做对了？"的质疑感，嘴角微微抿住，不是生气而是"我有话要说" |
| **动作** | 右手掌心向前推出，做一个轻微的"停一下"手势，像在叫停观众。身体微微前倾          |
| **视线** | 看镜头，有直视对话感                                    |
| **穿着** | 根据实际拍摄当天穿着填写                                  |
| **背景** | 根据实际拍摄环境填写                                    |
| **构图** | 人物偏右，占画面 55%。左侧留白放文字                          |


### 封面文字排版

```
┌─────────────────────────┐
│                         │
│  别再这样               │  ← 白色加粗大字，左上方
│  问AI                   │  ← 亮黄色加粗大字，视觉重点
│                         │
│              ┌────────┐ │
│              │        │ │
│              │  人物   │ │  ← "停"手势，微蹙眉
│              │  (右侧) │ │
│              │        │ │
│              └────────┘ │
└─────────────────────────┘
```

- 文字放左侧和左上方，不遮挡人脸
- "问AI" 用亮黄色或橙色加粗，其余用白色
- 所有文字加深色描边/阴影，确保清晰
- 比例 3:4（小红书/抖音竖屏）

### 即梦AI提示词（参考）

> 小红书短视频封面图，3:4竖版比例。一位穿着[当天穿着]的年轻亚洲男性，背景为[当天背景]，室内温暖自然光线。他眉头微蹙，眼神带着质疑和认真，右手掌心向前推出做"停一下"的手势，像在叫停观众。身体微微前倾，看向镜头，有强烈对话感。人物位于画面右侧，左侧留出大面积空白。光影柔和自然，高真实感，色彩纯净简约。

> **注意**：文字不要让即梦生成，用醒图/剪映手动加。

### 醒图/剪映加字步骤

1. 选即梦生成的图（或拍摄时截取的一帧）
2. 左上方添加文字"别再这样"——白色，加粗，字号大
3. 下一行"问AI"——亮黄色或橙色，加粗，字号略大于上面
4. 所有文字加黑色描边或投影，确保可读性
5. 导出 3:4 比例

## 脚本正文

---

### ⏱ 开场钩子（约 5 秒）

> **核心任务**：一句话好奇缺口，让观众停下来
> **形式**：口播

你向AI提的每个问题，可能从第一句就错了。

---

### ⏱ 问题共鸣（约 20 秒）

> **核心任务**：描述大部分人的AI使用方式，触发"说的就是我"
> **形式**：口播 + 录屏穿插

你是不是也这样？想做一件事，打开AI就直接问，怎么做自媒体、怎么转行、怎么学英语。AI噼里啪啦给你列了一二三四五，你看完觉得好有道理，但关掉之后还是不知道具体该干嘛。

> **【插入录屏1】**：在 Cursor 中向 Claude 提问"帮我做一个自媒体定位方案"，AI输出一堆泛泛建议（"找到你的优势""确定目标受众""选择内容方向"等）
> 停留 10-12 秒，让观众看到这类"正确的废话"

---

### ⏱ 点出本质 + 引出方法（约 20 秒）

> **核心任务**：一句话点透问题本质，给出不同的用法
> **形式**：口播

其实问题不在AI的回答，在你的问法。你在问"怎么做"，但你自己都还没想清楚"你是谁、你的目的是什么"。换一种方式—别问AI答案。把你真实的情况、你的经历、你的困惑全部丢给AI，让AI反过来引导你，一层一层帮你从底层开始想清楚。

---

### ⏱ 真实案例——起点（约 15 秒）

> **核心任务**：铺垫"一团浆糊"的起点，制造前后反差
> **形式**：口播

我自己就是这么过来的。刚开始做自媒体的时候，看到搞笑博主、情侣博主、带货博主接各种广子，我也想做。但我完全没想过—我自己有什么优势？我能帮别人解决什么问题？结果呢？发了快20条视频，才涨了不到100个粉丝。

---

### ⏱ 真实案例——转折（约 25 秒）

> **核心任务**：展示"让AI引导思考"的真实过程
> **形式**：口播 + 录屏穿插

后来我换了种方式。我没有问AI"怎么做自媒体定位"，我把我真实的情况全部丢给了AI——我的工作背景、我做过什么、我擅长什么、我纠结什么。AI没有直接告诉我该做什么，而是反过来一直在追问，你的个人背景是什么？你有什么优势？你想做什么内容？目标受众是谁？怎么变现？

> **【插入录屏2】**：在 Cursor 中展示与 Claude 的深度对话——创作者描述自己的背景和困惑，Claude 在追问和层层拆解（如"你的个人背景是什么？""你有什么优势？""目标受众是谁？"）
> 停留 10-15 秒，重点展示 AI 在"反问"而不是"回答"

---

### ⏱ 真实案例——结果（约 25 秒）

> **核心任务**：展示从混乱到清晰的结果，强化反差
> **形式**：口播 + 录屏/截图穿插

你在互联网公司做了8年运营，运营是我的优势没错，但运营是你在教育公司做的事，那教育，才是你真正的根。就这一句话点醒了我。我之前一直觉得自己是个"做运营的"，但AI帮我看到了更底层的东西，8年教育行业的经验，才是我真正稀缺的资产。

几轮对话下来，从一团浆糊，变成了一份完整的方案——我的定位、内容方向、变现路径、过渡策略，全部清清楚楚。

> **【插入录屏3/截图】**：展示 IP定位分析文档的结构——标题、目录、定位方案、策略演变等，让观众直观感受"从混乱到结构化"的变化
> 停留 10-12 秒，展示文档的完整度和条理性

---

### ⏱ 升华 + 行动点（约 20 秒）

> **核心任务**：金句收尾 + 强化单一行动点
> **形式**：口播

AI最大的价值，不是给你一个标准答案，是帮你问对问题、想清楚你自己。

下次你想做一件事但脑子里一团乱的时候，别问AI"怎么做"——把你的真实情况告诉AI，让它来引导你。答案一直在你自己身上，AI只是帮你把它挖出来。
