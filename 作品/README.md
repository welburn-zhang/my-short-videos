# 作品目录索引

一条内容一个目录，命名为 `YYYYMMDD_选题简称/`。日期一般用开始创作或建目录的日期，不强求等于发布时间。

## 使用约定

- 新增选题后，把对应行插在全量索引最上方。
- `定稿.mov` 只能说明本地存在成片，不直接等于平台已发布。
- 如果某条内容进入复盘、翻新或素材提取阶段，优先在作品目录内补 `数据观察.md` 或 `选题说明.md`。
- 当前不移动历史目录；先靠索引降低查找成本。
