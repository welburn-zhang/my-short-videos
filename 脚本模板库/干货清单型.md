# 脚本模板：干货清单型

> 适用于"N个方法/习惯/误区"这类内容。节奏快，信息密度高，收藏率高。
> 参考案例：`作品/20260322_学习效率3个习惯/脚本.md`

---

## 适用场景

- 总结N个方法/习惯/工具/误区
- 适合做成图文+视频双发（小红书效果尤其好）
- 收藏率高，但互动率相对低——适合积累收藏，不适合冲评论

---

## 脚本结构

```
① 钩子（约10秒）：抛出「N个X」+ 为什么值得看
② 身份背书（约10秒）：8年教育经验/真实观察
③ 清单正文（约40-60秒/每条）：
   - 每条结构：[问题/现象] → [本质原因] → [具体做法] → [可执行动作]
④ 结尾收口（约15秒）：总结 + 今天选一个试试
```

---

## 填空模板

### 发布信息表

| 项目 | 填写 |
|------|------|
| 主题 | [N]个[主题] |
| 栏目线 | AI帮你开始 / AI帮你学会 / AI帮你做完 |
| 数量 | 3个（不超过4个，太多了记不住） |
| 预计时长 | 约 3-4 分钟 |

---

### 封面 / 标题 / 标签

**小红书封面文案**（收藏引导型）：「[N]个[主题] / [具体收益]」

**小红书主标题**（≤20字）：[身份背书]+[N]个[主题]

**抖音封面文案**：「[最反常识的一条]」

**抖音主标题**：在[行业]做了[N年]，[反常识结论]
