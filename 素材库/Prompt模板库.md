# Prompt 模板库

> 收录实测有效的 AI 提示词，按使用场景分类。
> 格式：场景 + Prompt 正文 + 注意点 + 来源

---

## 一、学习类

### 1.1 拆第一步（降低启动门槛）

**场景**：想学一个新东西但不知道从哪开始，或者老是学两天就断

**Prompt**：

```
我想学___。
我现在是___水平。
别给我完整计划，只给我今天能做完的第一步，控制在10分钟内，并告诉我做到什么程度算完成。
```

**注意**：不要省略"做到什么程度算完成"，这句话能给出明确完成标准，防止"做了但不知道算没算完"的焦虑。可以把10分钟换成其他时长。

**来源**：AI拆第一步（选题13）

---

### 1.2 资料整理成一页

**场景**：读完一篇文章/看完一个视频，想快速提炼能用的东西

**Prompt**：

```
帮我把下面这段内容整理成一页：
- 3个最重要的重点（每条不超过2句话）
- 2个我需要注意的地方
- 1个明天就能用上的行动

内容：[粘贴内容]
```

**注意**："明天就能用上"这个限制很关键，强迫AI给出可执行结论而不是概述。

**来源**：资料看完记不住（选题14）

---

### 1.3 学完让AI考你

**场景**：学完一个知识点，想验证是否真的掌握了

**Prompt**：

```
我刚学完[___]，帮我出3道选择题测验。
要求：
- 每题4个选项，其中有1个明显对的，有1个看起来像对但其实错的
- 出完不要给答案，等我回答完你再评分
```

**注意**：错的题要回去补，不要死记答案。

**来源**：AI学习最关键一步（选题5）

---

### 1.4 用AI引导自我分析

**场景**：遇到很难想清楚的问题（定位、选择、规划），不知道怎么下手

**Prompt**：

```
我现在面临一个问题：[描述你的真实情况，包括背景、约束、你已经想到的]
不要给我建议，先帮我用3个追问把这个问题想清楚。等我回答完你再给建议。
```

**注意**：先描述真实情况，不要只问"怎么办"。让AI问你比让AI给答案更有用。

**来源**：向AI提问的正确方式（选题7）

## 二、工作提效类

### 2.1 排优先级

**场景**：任务一多不知道先做哪个

**Prompt**：

```
帮我排这些任务的优先级：
任务清单：[列出所有任务]
今天可用时间：[X小时]
限制条件：[如：有2点的会/明天有截止]
帮我按"今天必须完成 / 今天做更好 / 可以推到明天"分类，并告诉我今天最先做哪个。
```

**来源**：AI排优先级（选题15）

---

### 2.2 快速竞品/行业调研

**工具**：Gemini DeepResearch

**用法**：打开Gemini，开启DeepResearch，直接说：

```
帮我分析[行业/竞品]，重点是[你最关心的维度]。
输出格式：市场概况 + 主要玩家对比 + 关键趋势 + 对我的启示
```

**注意**：Gemini DeepResearch会自动搜几十个网站，不需要自己搜资料。

**来源**：AI职场提效3个场景（选题12）

---

### 2.3 AI改简历（正确姿势）

**场景**：想用AI优化简历，但不想改出来的东西全是废话

**Prompt**：

```
我要投[岗位名称]，以下是这个岗位的JD和我的简历：

JD：[粘贴JD]
我的简历：[粘贴简历]

不要帮我美化表达，先帮我找：
1. 我的经历里哪些最匹配这个JD
2. 哪些经历描述和JD要求有明显落差
3. 有哪些经历可以重新包装后匹配JD

找完再问我要不要改。
```

**注意**：先诊断再修改，不要上来就让AI帮改，改出来全是通用废话。

**来源**：AI改简历第一步就错了（选题17）
