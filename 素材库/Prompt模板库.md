# Prompt 模板库

> 收录实测有效的 AI 提示词，按使用场景分类。
> 格式：场景 + Prompt 正文 + 注意点 + 来源

---

## 一、学习类

### 1.1 学完让AI考你

**场景**：学完一个知识点，想验证是否真的掌握了

**Prompt**：

```
我刚学完[___]，帮我出3道选择题测验。
要求：
- 每题4个选项，其中有1个明显对的，有1个看起来像对但其实错的
- 出完不要给答案，等我回答完你再评分
```

**注意**：错的题要回去补，不要死记答案。

**来源**：AI学习最关键一步（选题5）

---

### 1.2 用AI引导自我分析

**场景**：遇到很难想清楚的问题（定位、选择、规划），不知道怎么下手

**Prompt**：

```
我现在面临一个问题：[描述你的真实情况，包括背景、约束、你已经想到的]
不要给我建议，先帮我用3个追问把这个问题想清楚。等我回答完你再给建议。
```

**注意**：先描述真实情况，不要只问"怎么办"。让AI问你比让AI给答案更有用。

**来源**：向AI提问的正确方式（选题7）

## 二、工作提效类

### 2.1 AI改简历（正确姿势）

**场景**：想用AI优化简历，但不想改出来的东西全是废话

**Prompt**：

```
我要投[岗位名称]，以下是这个岗位的JD和我的简历：

JD：[粘贴JD]
我的简历：[粘贴简历]

不要帮我美化表达，先帮我找：
1. 我的经历里哪些最匹配这个JD
2. 哪些经历描述和JD要求有明显落差
3. 有哪些经历可以重新包装后匹配JD

找完再问我要不要改。
```

**注意**：先诊断再修改，不要上来就让AI帮改，改出来全是通用废话。

**来源**：AI改简历第一步就错了（选题17）
