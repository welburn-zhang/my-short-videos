# 张半蛋人物形象参考

本目录保存张半蛋本人各角度形象照，用于后续短视频封面图生成时作为人物形象基底参考。

## 使用规则

- 生成短视频封面图前，默认先参考本目录的人物形象照，再结合具体选题和封面制作规则生成。
- 优先使用正面、半身、思考姿态作为封面人物基底；侧脸图主要用于补充五官、发型和脸型参考。
- 封面人物应保持亚洲男性、黑色短发、清爽自然、真实创作者气质。
- 人物表情可以有反应感，但不追求极度夸张；不要生成过度网红化、二次元化或与本人差异很大的形象。
- 生成图仍需遵守项目封面规则：高对比大字、清晰主题物件、搞笑/抽象/搞怪但不无厘头，真实任务和真实判断优先。
