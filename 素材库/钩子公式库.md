# 钩子公式库

> 收录**经过数据验证**的开头句式和钩子结构。
> 格式：公式类型 + 示例 + 实测反馈 + 适用场景。

---

## ⚠️ 素材库准入规则（铁律）

**本库不是灵感记录本，是经过数据验证的经验沉淀。新素材必须通过验证才能进库。**

- ✅ **准入门槛**：作品发布 **7 天后**，小红书/抖音两个平台各自满足以下三条中的至少两条：
  - 点赞/完播/每万观看涨粉 **≥ 账号同期均值 × 1.2**
  - 5 秒完播率 ≥ 40%（抖音硬门槛，过不了算法推荐就直接失败）
  - 评论 ≥ 5 条真实共鸣（不是"沙发""打卡"）
- ❌ **不得在发布当天/24h/72h 把新钩子写进本库**（数据没出来，都是主观感觉）
- 🧪 **数据不达标但有教训价值**：写进文末「反面教训」区，不要写进主区
- 🔄 **数据观察**：发布后在 `作品/{日期}_{标题}/数据观察.md` 里记录，7 天后回来决定升级/淘汰

## 使用原则（来自 dbs-hook 总结）

1. **开头必须独立工作**——不能假设用户看了标题/封面
2. 前5秒要有抓手：数据 / 金句 / 反差 / 痛点共鸣
3. 制造悬念，不直接给答案
4. 口播友好，能直接念出来，不要自问自答
5. **开头公式 = 话题（讲什么）+ Hook（为什么看）+ 可信度（为什么信你）**

## 公式一：反转否定型「不是X，是Y」⭐⭐⭐⭐⭐

**结构**：否定用户常见的自我归因 → 给出更真实的解释 → 触发共鸣

**模板**：

```
[痛点场景]，很多人[常见错误归因]。
但很多时候，不是[归因]，是[真正原因]。
```

**实测案例**：

- "学新东西老是学两天就断，很多人第一反应：我又不自律了。但很多时候，问题不是你没毅力，是你给自己的第一步太大了。" → **AI拆第一步**
- "资料看了很多，脑子里只剩'我记得我看过'。很多人以为这是记性差。其实不是，是你从头到尾都没整理过。" → **资料看完记不住**
- "AI老回废话？不是AI不行，是第一句只说了半句话。" → **AI第一句怎么说**

**适用栏目**：AI帮你学会、AI帮你开始、AI帮你做完 均适用

## 公式二：痛点场景化「你有没有这种感觉」⭐⭐⭐⭐

**结构**：具体还原用户经历过的失败场景 → 不急着给解法 → 共鸣后才给方案

**模板**：

```
你有没有这种感觉，[具体描述失败场景]，[尴尬/难受的结果]？
```

**实测案例**：

- "你有没有这种感觉，资料看了不少，收藏也存了一堆，过两天别人一问，你脑子里只剩一句：我记得我看过。" → **资料看完记不住**
- "事情一多脑子就乱，明明忙了一整天，最重要的事还是没动。" → **AI排优先级**

**注意**：节奏较慢，**小红书用，抖音慎用**（抖音前3秒要更快）

---

## 公式三：数字冲击型「X%的人/X年」⭐⭐⭐

**结构**：用数字引出反常识 → 制造好奇

**模板**：

```
[X]% 的人[行为/状态]，[反常识结论或悬念]。
```

**实测案例**：

- "用AI学东西，90%的人漏了最关键一步" → **AI学习最关键一步，效果好**
- "面试了几百人，这道题几乎全军覆没" → **面试最大的缺点**

**⚠️ 教训**：**开头不要连续堆多个数字**（"会AI涨薪8-10%，不会只涨2-3%，超3成岗位..."）→ 5s完播率只有22%。一个数字够了。

## 公式四：身份背书型「我在X做了N年」⭐⭐⭐

**结构**：建立可信度 → 引出观察/教训

**注意**：**不适合放第一句**。适合放在痛点之后的第3-5秒做背书。

**模板**：

```
我在[行业]做了[N年]，[这种情况/这个问题]我见过太多了。
```

**实测案例**：

- "我在教育行业做了8年，这种情况我见过太多了。很多孩子不是不想学，是一上来任务就太重。" → 接在痛点场景后，背书自然

---

## 公式五：直接陈述型「[有反差的事实]」⭐⭐⭐⭐

**结构**：直接说一个让人想继续听的事实/判断

**注意**：不要「你以为...？其实...」这种自问自答，念起来不自然

**实测案例**：

- "很多人觉得AI不好用，是因为问了半天，回来还是一堆废话。" → 简洁，口播友好
- "学东西总是学两天就断，多数时候不是没毅力，是第一步给自己的任务太重了。" → 直接陈述判断

## 抖音专用：3秒加速版开头

在正片前加一段3-5秒的快版，直接抛出最强结论：

```
[核心结论的一句话版本]？[反转/答案/悬念]。
```

**示例**：

- "学新东西总是学两天就断？很多时候不是你没毅力，是第一步太大了。" → 接正片
- "资料总记不住？不是记性差，是没整理。" → 接正片
- "AI老回废话？不是AI不行，第一句就说错了。" → 接正片
