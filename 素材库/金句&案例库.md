### #开始

- "人更容易坚持的，从来不是那种听起来很厉害的计划，而是今天真的做得完的小动作。" ← AI拆第一步
- "先开始，比先计划更重要。" ← AI拆第一步
- "不是你懒，是第一步太大，刚开始就把自己劝退了。" ← AI拆第一步
- "做完了也不知道算不算完成，心里就没有那种'我今天真推进了一点'的感觉。" ← AI拆第一步

### #学习

- "当下会觉得懂了，过几天基本就忘了——不是记性差，是看完之后没有整理成自己的表达。" ← 资料看完记不住
- "看懂不等于记住，输入完还需要输出。" ← AI学习最关键一步
- "收藏了很多，脑子里只剩一句：我记得我看过。" ← 资料看完记不住（可直接用在开场）

### #AI使用

- "不是AI不行，是第一句只说了半句话，没把真实情况交代清楚。" ← AI第一句怎么说
- "问题太大，AI就只能回你一堆谁都没法马上用的废话。" ← AI拆第一步
- "别问AI答案，把真实情况丢给AI，让AI反过来引导你思考。" ← 向AI提问的正确方式
- "很多人用AI停留在一问一答，本质上还是高级搜索引擎。" ← 80%的人不会用AI

### #执行

- "明明忙了一整天，最重要的事还是没动——问题往往不是效率差，而是一开始没先分轻重。" ← AI排优先级

### #教育经验

- "任务越清楚、越小，人越容易进状态。先把第一步做出来，后面才谈得上坚持。" ← AI拆第一步（8年教育观察）
- "很多孩子不是不想学，是一上来任务就太重，刚开始就烦了。成年人也一样。" ← AI拆第一步（教育案例）
- "成绩好的孩子，大多不是天赋更好，而是学完当天会自己考自己。" ← 学习效率3个习惯

## 案例区

### #教育行业真实案例

- **一人AI顶7人**：用AI一周完成了本来需要7个人两周才能完成的活动策划工作。← 8年运营大实话
- **AI改简历**：大多数人打开简历第一件事是让AI帮改，但方向从第一步就错了——应该先搞清楚投的是什么岗、对方想要什么人。← AI改简历第一步就错了
- **定位案例**：创作者做了50条视频只涨了不到100粉，让AI用第一性原理深挖，发现"教育才是8年经验的根"，完整输出了定位方案。← 向AI提问的正确方式

### #AI学习真实案例

- **学SOTA**：学SOTA概念 → 让AI出结构化笔记 → AI出4道测验题全对，从"觉得懂了"变成"真的会了"。← AI学习最关键一步
- **AI读7份报告**：用AI读完7份行业报告 → 结构化总结 → 输出职业转型规划，从信息堆到可用决策。← AI时代什么人最值钱
