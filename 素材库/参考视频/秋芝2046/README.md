## 提炼规则

- 开头不是先解释概念，而是先给强判断、强结果或强情绪。
- 3-15 秒内完成：轻松打招呼或直接切入 → 关键词放大 → 生活化画面 / 荒诞比喻 → 说清这条能解决什么。
- 画面上要有大字关键词或强视觉包装，不能只靠口播解释。
- 语气轻松但认真，好笑来自比喻和画面，不靠夸张表演。
- AI 工具、Agent、工作流、创作过程类视频后续默认复用这一套开头方法。

## 适用于

- Codex / Agent / Skill / MCP / 插件 / 自动化 等 AI 工具类内容。
- 工作流、创作过程、AI 帮你做完类内容。
- 需要把抽象概念讲给普通人听的内容。

## 不适合

- 直接定义概念。
- 上来念产品功能。
- 说「今天我们来讲……」。
- 做安装、登录、配置、官网入口教学。
