## ⚠️ 素材库准入规则（铁律）

**本库不是"所有发过的封面都要记"，只记录已经跑出数据的成功模板。**

- ✅ **准入门槛**：作品发布 **7 天后**，该封面对应的作品在小红书/抖音各自满足以下三条中的至少两条：
  - 点击率（如平台可见） **≥ 账号同期均值 × 1.2**
  - 点赞/完播/每万观看涨粉 **≥ 账号同期均值 × 1.2**
  - 5 秒完播率 ≥ 40%（抖音硬门槛）
- 📋 **「已发布数据记录」表**：所有发过的封面都可以先录入本表做观测，但**不等于升级成公式**
- ❌ **新公式不得在发布当天/24h/72h 归纳写入本库**（感觉好≠真的好）
- 🧪 **数据不达标的封面**：在「已发布数据记录」表里标 ❌，文末「反面教训」区简要说明为什么没跑起来
- 🔄 **数据观察**：发布后在 `作品/{日期}_{标题}/数据观察.md` 里记录，7 天后回来决定升级/淘汰

## 使用原则

- 封面默认**不把「AI」当主卖点**，优先卖痛点、好奇心、冲突或具体收益
- 小红书标题 ≤ 20 个字，优先短句、痛点或反常识
- 抖音标题优先解决前3秒留存，给痛点/冲突/场景
- 封面风格轮换：**对话感 / 冲突对比 / 干货清单感**（每3-4条穿插，不要每条都用"正脸+大号彩色文字"）

## 公式一：「不是X / 是Y」冲突对比型

**说明**：反转用户归因，两行文字形成冲突，视觉张力强


| 小红书封面              | 抖音封面            | 来源内容     |
| ------------------ | --------------- | -------- |
| 「不是记性差 / 是你没整理」    | 「不是记性差 / 是你没整理」 | 资料看完记不住  |
| 「学两天就断？/ 别急着怪自己」   | 「别再做计划 / 先拆第一步」 | AI拆第一步   |
| 「AI总回废话？/ 你第一句没说全」 | 「别让AI猜 / 先交代清楚」 | AI第一句怎么说 |


---

## 公式二：「[痛点]？/ 先[动作]」行动引导型

**说明**：把问题和解法各压一行，解法要具体可操作


| 小红书封面             | 抖音封面                  | 来源内容    |
| ----------------- | --------------------- | ------- |
| 「资料总记不住？/ 先整理成一页」 | 「资料总记不住？/ 不是记性差，是没整理」 | 资料看完记不住 |


---

## 公式三：数字 + 人群型

**说明**：用数字制造权威感或反常识


| 封面文案              | 适用平台 | 来源内容      |
| ----------------- | ---- | --------- |
| 「90%的人漏了这一步」      | 双平台  | AI学习最关键一步 |
| 「8年教育行业 / 3个学习习惯」 | 小红书  | 学习效率3个习惯  |

## 标题公式

### 小红书标题（≤20字）


| 公式               | 示例              |
| ---------------- | --------------- |
| `[痛点]？先[解法]`     | 资料总记不住？先整理成一页   |
| `总[负面行为]？不是X是Y`  | 总坚持不下去？不是没毅力是这个 |
| `[身份]+[场景]+[数字]` | 教育行业8年看到的3个学习习惯 |
| `[反常识结论]`        | 向AI提问，你的方式可能全错了 |


### 抖音标题（强钩子优先）


| 公式                    | 示例                   |
| --------------------- | -------------------- |
| `[负面场景]？不是X，是你[具体原因]` | 学两天就放弃？不是没毅力，是第一步走大了 |
| `[痛点]？[身份背书]+[核心解法]`  | 资料总记不住？不是记性差，是没整理    |
| `AI[问题]？[第一步错在哪]`     | AI老回废话？第一句先别这么问      |
| `[数字]+[反常识]`          | 90%的人用AI学习都在浪费时间     |
