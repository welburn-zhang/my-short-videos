# 学习笔记体系

## 目录说明

```
学习笔记/
├── README.md                  ← 你正在看的这个文件
├── 学习进度总览.md              ← 所有学习记录汇总 & 薄弱点追踪
│
├── 自媒体运营/                 ← 账号运营、涨粉策略、流量逻辑等
├── 内容创作/                   ← 脚本技巧、选题方法、爆款拆解等
├── AI工具应用/                 ← AI辅助创作、工具使用、提示词等
├── AI产品认知/                 ← 大模型原理、RAG、Agent、多模态、能力边界等
├── 个人成长/                   ← 职场认知、个人IP、思维方式等
├── 平台研究/                   ← 小红书/抖音/视频号 平台算法与玩法
│
└── 测验报告/                   ← 每次聊天后的测验 & 学习报告
    └── YYYYMMDD_主题_报告.md
```

## 使用流程

1. **日常聊天**：你向 AI 提问任何问题（运营、创作、AI工具、个人成长等）
2. **知识沉淀**：聊完后 AI 会整理核心知识点，归档到对应分类目录
3. **选择题测验**：AI 出 3-5 道选择题，考验你对本次内容的掌握程度
4. **生成报告**：根据你的作答情况，生成学习报告存入 `测验报告/` 目录
5. **进度追踪**：`学习进度总览.md` 持续更新，记录学习轨迹和薄弱环节

## 文件命名规范

| 类型 | 格式 | 示例 |
|------|------|------|
| 知识笔记 | `YYYYMMDD_主题关键词.md` | `20260314_短视频钩子设计.md` |
| 测验报告 | `YYYYMMDD_主题_报告.md` | `20260314_短视频钩子设计_报告.md` |

## 报告模板说明

每份测验报告包含：
- **学习主题**：本次聊天核心话题
- **知识要点回顾**：3-5 个核心知识点
- **测验题目与作答**：选择题 + 你的答案 + 正确答案 + 解析
- **得分与评价**：掌握程度评估
- **薄弱点标注**：答错的知识点，方便后续复习
- **下次建议**：针对薄弱点的学习建议
