## 知识要点回顾

1. SOTA = State of the Art，当前最先进水平
2. SOTA 针对具体任务，不是笼统的"最强"
3. SOTA 不等于最好用，跑分和实际体验是两码事
4. Benchmark 是标准化测试，SOTA 是测试上的最高成绩

### Q1: SOTA 的全称是什么？
- 你的答案：**B. State of the Art** ✅
- 解析：SOTA 是 State of the Art 的缩写，直译"当前最先进水平"

### Q2: 说某个模型是 SOTA，最准确的理解是？
- 你的答案：**B. 它在某个特定任务/基准测试上取得了当前最好成绩** ✅
- 解析：SOTA 是针对具体任务说的，不是所有方面都最强

- 解析：跑分第一不等于日常最好用，还要考虑速度、价格、稳定性等因素

### Q4: Benchmark 和 SOTA 的关系是？
- 你的答案：**B. Benchmark 是标准化测试，SOTA 是这些测试上的最高成绩** ✅
