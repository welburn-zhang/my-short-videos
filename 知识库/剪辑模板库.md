## 模板 A：纯口播

> **适用**：全程真人出镜说观点，没有录屏/外部素材
> **代表作**：春招简历致命错误、8年运营大实话

---

### A1. 剪映项目设置


| 项目   | 设置        |
| ---- | --------- |
| 画布比例 | 9:16（竖屏）  |
| 分辨率  | 1080×1920 |
| 帧率   | 30fps     |


---

### A2. 时间线结构（对照脚本段落）

```
[0:00 ─── 0:12] 开场钩子
[0:12 ─── 0:40] 第一个观点
[0:40 ─── 1:15] 第二个观点
[1:15 ─── 1:45] 第三个观点
[1:45 ─── 2:10] 加分技巧 / 重头戏
[2:10 ─── 2:20] 结尾引导互动
```

> 时长浮动正常，关键是**每段之间有节奏变化**，不要一镜到底的平淡感。

---

### A3. 剪辑操作清单（按顺序执行）

#### 第一步：粗剪（5 分钟）

1. 导入素材.mov
2. **开启「智能字幕」**→ 识别字幕 → 选「已校对」样式
3. 通篇听一遍，把**口误、卡壳、过长停顿**的部分直接裁掉
4. 每段观点之间，留 0.3-0.5 秒的自然呼吸间隔（不要完全无缝）

#### 第二步：字幕样式（3 分钟）


| 项目  | 设置                |
| --- | ----------------- |
| 字体  | 思源黑体 / 抖音美好体      |
| 字号  | 7-8（剪映单位）         |
| 颜色  | 白色，黑色描边（描边宽度 3-4） |
| 位置  | 画面下方 1/4 处（不挡脸）   |
| 对齐  | 居中                |


**关键词高亮规则**（手动调整）：

- 每段的**核心记忆点**改为黄色或橙色加粗
- 例："黑白色，从上到下，一栏到底" → 这句整体变黄
- 例："一个人，一周就做完了" → 变黄加粗
- 每段只高亮 1 句，不要满屏彩色

#### 第三步：重点文字贴片（5 分钟）

> 这是口播视频最关键的画面变化手段，让观众不走神。

**什么时候加文字贴片：**

- 每段观点的小标题（如"第一，简历太花了"）
- 关键数据/金句（如"5到10秒"、"JD是答题卡"）

**怎么加：**

1. 剪映 → 文字 → 新建文本
2. 字体：思源黑体 Bold
3. 字号：12-15（比字幕大）
4. 样式：白字 + 黑色阴影，或者黄字 + 黑色描边
5. 位置：画面上方 1/3（和字幕错开）
6. 动画：选「打字机」或「弹出」入场，持续 2-4 秒后消失
7. 每个文字贴片之间**至少间隔 10 秒**，不要太密

**参考节奏（以春招简历为例）：**


| 时间点  | 文字贴片内容           | 动画    |
| ---- | ---------------- | ----- |
| 0:12 | ❶ 简历太花了          | 打字机入场 |
| 0:30 | 黑白色 从上到下 一栏到底    | 逐字弹出  |
| 0:40 | ❷ 个人优势全是废话       | 打字机入场 |
| 1:05 | JD 是答题卡 你的优势就是答案 | 逐字弹出  |
| 1:15 | ❸ 一看就是海投         | 打字机入场 |
| 1:35 | 每投一个岗位 花 5 分钟    | 逐字弹出  |
| 1:45 | 💡 加分技巧          | 缩放弹出  |


#### 第四步：转场 & 跳切（3 分钟）

> 纯口播不需要花哨转场，靠**跳切**制造节奏感。

**跳切规则：**

- 每段观点切换时，直接硬切（不加转场特效）
- 如果同一段里有明显的语气转折，也可以跳切一下
- 跳切的本质：把拍摄时的停顿/思考时间剪掉，让节奏更紧凑

**唯一用转场的地方：**

- 开场钩子 → 第一个观点：可加一个「闪白」（0.3秒），暗示"正片开始"
- 正文 → 结尾：可加一个「叠化」（0.5秒），暗示"总结了"

#### 第五步：音效 & BGM（3 分钟）

**BGM：**


| 段落  | BGM 选择                       | 音量              |
| --- | ---------------------------- | --------------- |
| 全片  | 轻节奏感背景音乐（剪映搜"vlog轻快"或"知识分享"） | 10%-15%（不能盖过人声） |


> BGM 不要有歌词，不要太嗨，选一首循环到底就行。

**音效（剪映音效库直接搜）：**


| 触发时机        | 音效类型    | 搜索关键词      |
| ----------- | ------- | ---------- |
| 每段小标题出现时    | 提示音     | "提示音"、"叮"  |
| 关键金句出现时     | 强调音     | "强调"、"划重点" |
| 结尾互动引导时     | 轻快结束音   | "结束"、"完成"  |
| 说到反面案例时（可选） | 搞笑/失败音效 | "失败"、"哎呀"  |


> 音效不要超过 6-8 个/条视频，太多会吵。

#### 第六步：封面（2 分钟）

1. 暂停在一个表情自然的画面 → 截图
2. 剪映 → 图文 → 导入截图
3. 加封面文案（从脚本.md 的封面文案三选一）
4. 文字样式：大号加粗，白字黑描边，2-3 行排列
5. 确保小红书主页显示时文字不被裁切（安全区域内）

| --- | ----- |
| 分辨率 | 1080P |
| 帧率  | 30fps |
| 格式  | mp4   |

## 模板 B：录屏+口播

> **适用**：有工具演示、操作教程、对比展示的内容
> **代表作**：AI练面试靠谱吗、80%的人不会用AI

---

### B1. 素材准备（拍摄前）

你需要提前准备两类素材：


| 素材类型 | 要求                     | 文件                 |
| ---- | ---------------------- | ------------------ |
| 口播素材 | 正常竖屏拍摄，全程讲完            | 素材.mov             |
| 录屏素材 | 电脑/手机录屏，按脚本「录屏准备清单」逐条录 | 录屏1.mov、录屏2.mov... |


**录屏注意事项：**

- 电脑录屏：用 Mac 自带录屏（⌘+Shift+5）或 OBS，录完直接导入剪映
- 分辨率：1920×1080 即可（导入剪映后会适配竖屏）
- 字号调大：确保竖屏下文字清晰可读
- 鼠标操作放慢：让观众跟得上
- 关键内容在屏幕上停留 2-3 秒

---

### B2. 时间线结构

```
[0:00 ─── 0:12] 开场钩子 .............. 口播
[0:12 ─── 0:37] 实操演示 .............. 录屏 + 口播画外音
[0:37 ─── 1:07] 深度点评/分析 ......... 口播
[1:07 ─── 2:02] 核心干货/正确方法 ...... 口播 + 录屏穿插
[2:02 ─── 2:14] 补充提醒 .............. 口播
[2:14 ─── 2:24] 结尾引导互动 .......... 口播
```

---

### B3. 剪辑操作清单（按顺序执行）

#### 第一步：粗剪口播部分（5 分钟）

跟模板 A 一样：导入 → 智能字幕 → 去掉口误和卡壳

#### 第二步：插入录屏素材（8 分钟）

> 这是录屏+口播视频的核心操作。

**操作步骤：**

1. 对照脚本，找到需要切录屏的时间点
2. 在那个时间点把口播轨道「分割」
3. 把录屏素材拖入**画中画轨道**（不是替换主轨道）
4. 调整录屏素材大小和位置

**录屏画面的三种布局（按需选择）：**


| 布局              | 适用场景        | 设置方法                           |
| --------------- | ----------- | ------------------------------ |
| **全屏录屏 + 画外音**  | 需要观众仔细看操作时  | 录屏铺满全屏，口播音频保留但画面隐藏             |
| **上录屏 + 下口播**   | 边说边演示，两者都重要 | 录屏占上方 60%，口播缩小到下方 40%（圆形/方形裁切） |
| **口播为主 + 角落录屏** | 录屏只是辅助说明    | 口播全屏，录屏缩小到右上角（占 25%）           |


> 建议：演示操作步骤时用「全屏录屏」，讲解分析时用「口播为主」，两者交替切换制造节奏。

**录屏切入/切出的转场：**

- 口播 → 录屏：用「推入」或「闪白」（0.3秒）
- 录屏 → 口播：用「推出」或直接硬切

#### 第三步：录屏画面标注（5 分钟）

> 录屏画面信息量大，必须引导观众看哪里。

**标注工具（剪映 → 贴纸/文字）：**


| 标注类型    | 用法           | 搜索关键词       |
| ------- | ------------ | ----------- |
| 红色方框/圆圈 | 框住屏幕上的重点区域   | 贴纸搜"框"、"圈"  |
| 箭头指示    | 指向关键按钮/文字    | 贴纸搜"箭头"     |
| 放大镜效果   | 把小字内容放大让观众看清 | 关键帧 → 缩放该区域 |
| 文字标注    | 在录屏画面上加简短说明  | 新建文本，放在录屏上方 |


**标注节奏：**

- 录屏开始的前 1 秒：加一个标注指示"看这里"
- 关键操作时：红框/箭头跟随
- 每段录屏不超过 15 秒，超过就切回口播，再切回来

#### 第四步：字幕 & 文字贴片

跟模板 A 一样，但注意：

- 录屏全屏时，字幕位置改到**画面最底部**（避免挡录屏内容）
- 录屏画面上的文字标注用**黄色/红色**，跟白色字幕区分

#### 第五步：音效 & BGM

跟模板 A 基本一致，额外增加：


| 触发时机      | 音效     | 搜索关键词         |
| --------- | ------ | ------------- |
| 口播切到录屏时   | 切换音    | "切换"、"whoosh" |
| 录屏中点击/操作时 | 点击音    | "点击"、"tap"    |
| AI 输出结果时  | 成功/完成音 | "完成"、"成功"     |


#### 第六步：封面

跟模板 A 一样。如果内容涉及工具，可以在封面加一个工具界面的小截图增加信息量。

## 模板 C：横屏口播开场+穿插录屏

> **适用**：想做「口播开场 → 中间穿插录屏」、**整体横屏**、观众可点「全屏观看」的样式
> **对标参考**：@刘小呀（会用AI并不稀缺 / AI知识库等）

**和模板 B 的区别：**


| 维度    | 模板 B（竖屏）             | 模板 C（横屏）                |
| ----- | -------------------- | ----------------------- |
| 画布比例  | 9:16 竖屏              | **16:9 横屏**             |
| 发布后展示 | 竖屏信息流，上下有黑边或裁切       | 横屏信息流，平台会显示**「全屏观看」**按钮 |
| 开场    | 口播 / 录屏都可            | **一定是口播开场**，再切录屏        |
| 录屏段布局 | 多种（全屏录屏 / 上下分屏 / 角落） | 常用**录屏铺满 + 左下角口播小窗**    |


---

### C1. 剪映项目设置（横屏）


| 项目   | 设置           |
| ---- | ------------ |
| 画布比例 | **16:9（横屏）** |
| 分辨率  | 1920×1080    |
| 帧率   | 30fps        |


> 整条片子从拍到剪都按横屏来做，观众在抖音/小红书里会看到横屏画面 + 「全屏观看」入口。

---

### C2. 时间线结构（口播开场 → 穿插录屏）

```
[0:00 ─── 0:15] 口播开场（钩子） ......... 真人横屏出镜
[0:15 ─── 0:45] 第一段录屏演示 ........... 录屏铺满 + 左下角口播小窗（可选）
[0:45 ─── 1:10] 口播讲解/过渡 ............ 真人出镜
[1:10 ─── 2:00] 第二段录屏 / 多段录屏 .... 录屏为主 + 口播小窗
[2:00 ─── 2:20] 口播结尾 + 互动 ......... 真人出镜
```

**节奏要点：** 口播 → 录屏 → 口播 → 录屏，交替出现，避免长时间只有录屏或只有口播。

---

### C3. 拍摄与素材准备

**口播拍摄：**

- 手机/相机**横持**拍摄，人物居中或偏左/偏右，留出一点背景
- 背景简洁（如你截图里的室内白墙+画），光线均匀

**录屏：**

- 电脑横屏录 1920×1080，和项目分辨率一致，导入后不用缩放
- 深色界面（如 Cursor、编辑器）在横屏里更耐看
- 关键操作、关键文字在画面中轴附近，避免被裁切

---

### C4. 剪辑操作清单

#### 第一步：口播粗剪 + 字幕

1. 导入横屏口播素材，拖到主轨道
2. 智能字幕 → 识别 → 校对
3. 去掉口误、长停顿，口播段之间可跳切

#### 第二步：插入录屏段（口播 → 录屏的切法）

1. 在「要切到录屏」的那一帧，把口播轨道**分割**
2. 录屏素材拖到**主轨道**（接在口播后面），或拖到**画中画**和口播叠在一起（看你要「纯录屏」还是「录屏+口播小窗」）
3. **口播 → 录屏** 转场：用「推入」「闪白」或「叠化」0.3 秒，让切换明显
4. **录屏 → 口播** 同理，录屏段结束时切回口播轨道

#### 第三步：录屏段的布局（录屏为主 + 左下角口播小窗）

当某一段是「录屏为主、人声继续」时：

1. **主轨道**：录屏铺满 16:9 画布
2. **画中画**：口播片段拖进来，缩小到约 15% 画面
3. 位置拖到**左下角**（不挡录屏主体）
4. 画中画 → 蒙版 → **圆角矩形**，边缘更柔和；可加 2px 白描边
5. 口播的**声音**保留，画面只作为小窗，这样观众既能看操作又能看到你在讲

**字幕位置（横屏）：**

- 口播段：字幕在画面**下方**，不挡脸
- 录屏段：字幕在**最底部**或**左侧竖条**，避免挡界面关键信息

#### 第四步：录屏上的文字标注（黄色步骤条）

1. 文字 → 新建文本，写当前步骤（如「生成补充学习文件」）
2. 字体加粗，**黄色**，可加半透明黑色底条
3. 放在录屏画面**底部**或**左下角**（与口播小窗错开）
4. 每换一个操作就换一条标注，让观众知道「现在在看什么」

#### 第五步：音效与 BGM

- 口播 ↔ 录屏 切换时：可加 whoosh/切换音效
- BGM 轻量，不压人声（同模板 A/B）
- 录屏里关键操作可加点击/完成音效

#### 第六步：封面与导出

- 封面从口播或录屏里截一帧横屏图，加标题文案
- 导出：**1920×1080，16:9**，发布后平台会识别为横屏，出现「全屏观看」

---

## 通用素材包

> 以下设定适用于所有视频，只设置一次，后续作为「剪映草稿模板」复用。

---

### 字幕预设（保存为剪映模板）


| 项目  | 常规字幕         | 高亮字幕      | 文字贴片      |
| --- | ------------ | --------- | --------- |
| 字体  | 思源黑体 Regular | 思源黑体 Bold | 思源黑体 Bold |
| 字号  | 7            | 7         | 12-15     |
| 颜色  | 白色           | 黄色/橙色     | 白色或黄色     |
| 描边  | 黑色，宽度 3      | 黑色，宽度 3   | 黑色，宽度 4   |
| 阴影  | 无            | 无         | 有（增加层次）   |
| 位置  | 下方 1/4       | 下方 1/4    | 上方 1/3    |


---

### BGM 收藏夹（在剪映里收藏这些类型）


| 场景        | 风格关键词                   | 音量     |
| --------- | ----------------------- | ------ |
| 知识分享/观点输出 | "轻快vlog"、"知识分享"、"日常bgm" | 10-15% |
| 走心故事/大实话  | "温暖"、"治愈"、"钢琴轻音乐"       | 8-12%  |
| 工具教程/操作演示 | "科技感"、"电子轻音乐"           | 10-15% |


> 每种类型收藏 2-3 首备用，避免每次都重新找。

## 脚本级剪辑标注体系

> 口播脚本中直接标注剪辑动作，拍完对照标注剪辑，不用临场再想「这里该怎么处理」。
> 首次应用于「学习效率3个习惯」脚本V3，后续所有脚本统一使用此标注。

### 标注符号说明


| 符号  | 名称  | 含义                     | 剪映操作                    |
| --- | --- | ---------------------- | ----------------------- |
| ✂️  | 跳剪  | 前后各剪掉停顿，直接拼接           | 切片段拼接，不加转场              |
| 🔤  | 花字  | 大号文字压屏，强调重点            | 文本→新建文本，加粗白色+黑色描边       |
| 🔍  | 微变焦 | 画面推近（100%→115%），说重点时拉近 | 关键帧（起始100% → 0.3s后115%） |
| 🏷️ | 角标  | 屏幕角落的编号/标签             | 贴纸或固定文本框                |
| 🔊  | 音效  | 配合画面的提示音/强调音           | 音效库搜对应关键词               |


### 4 个核心手法（口播视频专用）

**1. 跳剪（节奏紧凑）**

- 每句话之间的停顿、口误、思考全部剪掉
- 观众没有走神的窗口
- 段落切换时直接硬切，不加花哨转场

**2. 花字（视觉抓重点）**

- 核心观点以大号文字压在画面上
- 不听声音也能抓到重点（很多人在地铁上刷视频不开声音）
- 每 15-20 秒一条花字，持续给视觉刺激
- 花字文案在脚本中直接写好，剪辑时照搬
- 列表式花字（如三种方法、三句话模板）多留 2-3 秒，方便观众截图

**3. 微变焦（靠近说重点）**

- 说到重点时画面微微推近（约110%-120%），制造「靠过来跟你讲」的感觉
- 说完重点回到正常景别，用推近/拉远的交替避免视觉疲劳
- 不是每句话都推，只在核心观点、金句、转折处使用

**4. 章节角标（进度感降低划走率）**

- 多要点内容（如「3个习惯」「5个特征」），在画面左上角放编号角标（❶ ❷ ❸）
- 让观众知道当前进度，降低中途划走冲动（「还有一个，看完再走」）
- 白底黑字小方块样式，从该章节开头持续到下一章节开始前消失
- 章节切换时配一个轻「叮」音效

### 脚本中的标注写法示范

```markdown
第一个，学完东西，马上试着讲出来。

> ✂️ 跳剪：这句单独一个镜头，说完直接接下句
> 🔍 微变焦：从正常景别推近
> 🔤 花字：**「学完马上讲出来」** 大字居中弹出

成年人也一样。你刷了一堆干货视频...你根本说不清楚。

> 🔍 微变焦：说到「你根本说不清楚」时推近
> 🔤 花字：**「看懂 ≠ 记住」** 大号白字 + 黑色描边，停留 2 秒
> 🔊 音效：花字弹出时加一个轻「咚」
```

### 花字样式速查


| 花字类型 | 用法          | 样式                    |
| ---- | ----------- | --------------------- |
| 核心观点 | 每个要点的标题句    | 大号白字居中弹出              |
| 金句总结 | 每段结尾的一句话总结  | 大号白字 + 黑色描边，停留 2 秒    |
| 操作列表 | 多个方法/步骤逐条展示 | 列表式逐条弹出，全部显示后多留 2-3 秒 |
| 对话示范 | AI 话术、提问模板  | 聊天气泡样式（剪映文本模板）        |
| 填写示范 | 模板的实际填写例子   | 手写笔记/便签样式，停留 3 秒      |
| 行动指令 | 结尾的号召行动     | 黄色大字强调弹出              |

## 剪辑节奏速查表

> 剪辑的本质就是控制节奏。以下是"什么时候该做什么"的速查。


| 脚本段落                | 画面该做什么           | 声音该做什么       |
| ------------------- | ---------------- | ------------ |
| **开场钩子**            | 保持口播画面不动，让观众聚焦内容 | 无音效，BGM 淡入   |
| **段落切换**（"第一"→"第二"） | 跳切 + 文字贴片入场      | 提示音          |
| **说到关键金句**          | 字幕变黄 + 可选文字贴片    | 强调音效         |
| **说到反面案例**          | 表情/语气自然有变化就够了    | 可选搞笑音效       |
| **口播→录屏**           | 闪白/推入转场          | whoosh 音效    |
| **录屏→口播**           | 硬切或推出            | 无            |
| **结尾互动引导**          | 口播画面 + 关注引导贴纸    | BGM 淡出 + 结束音 |

| 步骤         | 纯口播         | 录屏+口播       |
| ---------- | ----------- | ----------- |
| 粗剪（去口误、跳切） | 5 分钟        | 5 分钟        |

| 文字贴片       | 5 分钟        | 5 分钟        |

## 剪映效率技巧（Mac 电脑版）

1. **保存草稿模板**：把字幕样式、BGM、音效都设好的空项目保存为草稿，每次新建直接「复制草稿」
2. **智能字幕先行**：永远第一步开字幕，后续所有操作都基于字幕时间线来定位
3. **批量复制文字样式**：设好一个文字贴片的样式后，右键该文字 → 复制，再粘贴到时间线，只改文字内容即可
4. **标记功能**：在需要加音效/贴片的位置按 `M` 打标记，最后统一处理
5. **导出前检查**：全片播放一遍（空格键），重点听音量平衡（人声 > 音效 > BGM）

**Mac 剪映常用快捷键：**


| 操作       | 快捷键                |
| -------- | ------------------ |
| 分割       | `B` 或 `⌘B`         |
| 播放/暂停    | `空格`               |
| 撤销       | `⌘Z`               |
| 导出       | `⌘E`               |
| 放大/缩小时间线 | `⌘ + / ⌘ -` 或触控板双指 |
| 选中素材后删除  | `Delete`           |


> 更多快捷键：剪映 Mac 版 → 菜单栏「帮助」→ 快捷键说明
