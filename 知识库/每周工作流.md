# 每周工作流

> 每天 20:00-22:30，每周产出 3-4 条高质量视频

---

## 核心原则

1. **批量化**：同类任务集中做，不要一天内又写又拍又剪
2. **流水线**：周一写脚本 → 周二三拍摄 → 周四五剪辑 → 周末发布+运营
3. **不完美主义**：拍摄 3 遍以内搞定，不追求完美，真实感 > 精致感

---

## 每周时间表

### 周一：选题 + 写脚本（2.5h）

```
20:00-20:30  选题确认（30min）
             - 从选题池里挑本周要做的 3-4 个选题
             - 确认内容配比：1条求职 + 1条AI×运营 + 1条成长/人设
             - 如果选题池空了，花 15 分钟翻粉丝私信/评论/小红书搜索补充

20:30-22:30  AI 辅助写脚本（120min）
             - 用 AI 出初稿（每条约 15 分钟）
             - 自己改到「像自己说话」的状态（每条约 25 分钟）
             - 目标：一晚上写完 3-4 条脚本
```

**本日产出**：3-4 条脚本定稿 + 对应的封面文案/标题/话题标签

### 周二：拍摄日 1（2.5h）

```
20:00-20:15  准备（15min）
             - 换衣服、整理背景、调光线和机位
             - 把脚本过一遍，心里有数就行，不用背

20:15-21:45  拍摄 2-3 条视频（90min）
             - 每条拍 2-3 遍，选最自然的那一遍
             - 口播类每条实际拍摄 15-20 分钟
             - 模拟面试类需要换角度/换状态，预留 30 分钟

21:45-22:30  素材整理（45min）
             - 把素材导入手机/电脑
             - 按选题文件夹归档（直接放到 作品/YYYYMMDD_选题名/）
             - 标记每条素材的最佳 take
```

**本日产出**：2-3 条视频原始素材

### 周三：拍摄日 2 / 剪辑日 1（2.5h）

```
如果周二拍完了所有素材 → 直接进入剪辑：

20:00-22:30  剪辑 2 条视频（150min）
             - 每条剪辑约 60-70 分钟
             - 剪映流程：粗剪→加字幕→调节奏→加音乐→导出
             - 导出成品放到对应选题文件夹

如果周二只拍了 2 条 → 先补拍再剪辑：

20:00-20:45  补拍剩余 1-2 条（45min）
20:45-22:30  剪辑 1-2 条视频（105min）
```

**本日产出**：1-2 条成品视频

### 周四：剪辑 + 封面（2.5h）

```
20:00-21:30  剪辑剩余视频（90min）
             - 完成所有视频的剪辑和导出

21:30-22:30  制作封面图（60min）
             - 每条视频做 1 张封面
             - 可以用剪映/醒图/Canva
             - 大字 + 痛点关键词 + 清晰的人脸
             - 封面图存到对应选题文件夹
```

**本日产出**：所有成品视频 + 封面图完成

### 周五：发布第 1 条 + 运营（2h）

```
20:00-20:30  发布第 1 条视频（30min）
             - 小红书 + 抖音同步发布
             - 写好标题、话题标签、正文引导语
             - 评论区自己留 1 条引导互动的评论

20:30-21:30  运营工作（60min）
             - 回复本周所有评论和私信
             - 收集粉丝问题，记录到选题池
             - 看一下本周数据，简单记录

21:30-22:00  下周选题预想（30min）
             - 翻一下选题池，想想下周做什么
             - 有灵感就记下来，没有也不强求
```

### 周六：发布第 2-3 条（1.5-2h）

```
12:00        发布第 2 条（小红书中午发，抖音晚上发）
20:00-20:30  发布第 3 条
20:30-21:30  回复评论 + 运营互动
             - 如果有咨询私信，及时回复
             - 有精力的话刷一下同领域热门内容找灵感
```

### 周日：弹性时间（0-2h）

```
三种选择，看情况：

选项 A：休息，什么都不做（避免倦怠）

选项 B：发布第 4 条（如果本周产出了 4 条的话）

选项 C：做变现相关的事
         - 准备 1 对 1 辅导的材料
         - 整理资料包产品
         - 维护私域社群
```

## 每条视频的标准制作流程

```
第 1 步：写脚本（40min）
├── AI 出初稿（15min）
├── 自己改口语化（15min）
└── 确定封面文案 + 标题 + 标签（10min）

第 2 步：拍摄（20min）
├── 准备（5min）
├── 拍 2-3 遍（10min）
└── 检查素材确认 OK（5min）

第 3 步：剪辑（60min）
├── 粗剪：去掉废话和 NG 段（15min）
├── 加字幕（15min，剪映自动识别+手动校对）
├── 调节奏：卡点、停顿、重点强调（15min）
└── 加背景音乐 + 导出（15min）

第 4 步：封面 + 发布（20min）
├── 做封面图（10min）
├── 写发布文案 + 标签（5min）
└── 两个平台同步发布（5min）

单条总耗时：约 2.5 小时
```

## 批量化提效技巧

### 1. 脚本批量写

周一晚上一口气写完本周所有脚本。用 AI 辅助的话，3-4 条脚本 2 小时可以搞定。不要今天写一条明天写一条——切换成本很高。

### 2. 拍摄批量拍

一次换好衣服、调好灯光，连续拍 2-3 条。如果内容允许的话，可以换一件上衣就当换了一天。模拟面试类的可以集中拍同一天，效率最高。

### 3. 剪辑批量剪

先把所有视频粗剪完，再统一加字幕，再统一调节奏。不要一条视频从头做到尾再开始下一条。

### 4. 封面批量做

所有视频剪完后，统一做封面。保持封面风格一致，可以先做一个模板，后续只改文字。

## 选题池管理

**选题来源（按优先级排序）：**

1. 粉丝私信/评论中的真实问题（最优先）
2. 小红书搜索「运营面试」「产品运营」看用户在问什么
3. 自己 8 年运营经历中的真实故事和感悟
4. AI × 运营的新工具、新方法、新思考
5. 同领域博主的爆款选题（参考角度，不抄）

**选题池维护规则：**

- 想到就记，不论好坏，记到 `知识库/选题池.md`
- 每周一从池子里挑选，用不上的不删，留着以后用
- 目标：池子里始终保持 10+ 条待做选题，避免「周一坐下来不知道做什么」

---

## 每周检查清单

每周日晚上花 5 分钟过一遍：

- 本周发了几条？（目标 3-4 条）
- 内容配比对不对？（求职 35% / AI运营 30% / 成长人设 35%）
- 有没有回复所有评论和私信？
- 选题池里还有多少待做选题？（< 5 条就该补充了）
- 下周的选题想好了吗？
